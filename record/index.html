Adoption in software

Mikuture

Mikuture is a mobile AR application of Hatsune Miku, which hits million Downloads in 2015.

Yayoi Sakura has provided this fantastic AR application with three models(of cource “Appearance Miku” remodelings).

  • Frilled swim suit model
  • School uniform model
  • One piece model (specially designed for this application)



Co-operation with events

Bokurano Mirai (Our Future)

BokuMira (Bokurano Mirai) is a fan-made live concert event of Hatsune Miku, using hologram view technique. They have adopted my models from their first event in 2015.

Yayoi Sakura has provided them with “Alice costume edition” models specially developed for use in this event in 2016.



Miraino Neiro (Future Tone)

Miraino Neiro is also a fan-made live concert event of Hatsune Miku. It has an international activity basis.

Yayoi Sakura has provided them with “chinese costume edition” models specially developed for use in that event in 2016.
